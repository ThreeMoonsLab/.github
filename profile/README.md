# Three Moons Lab

Three Moons Lab builds developer infrastructure for agent release readiness and lifecycle health.

---

## What we ship

### [agents-shipgate](https://github.com/ThreeMoonsLab/agents-shipgate) · flagship

Static release-readiness scanner for AI agent tool surfaces — MCP, OpenAPI,
OpenAI Agents SDK, Google ADK. CLI + GitHub Action. Reviews what tools an
agent can call before it gets production-like tools or higher permissions.

> Static by default · no agent execution · no tool calls · no LLM calls ·
> no MCP connections · no network · no telemetry

→ [github.com/ThreeMoonsLab/agents-shipgate](https://github.com/ThreeMoonsLab/agents-shipgate)

---

## Principles

- **Static by default** — no agent execution at review time
- **Local-first** — no telemetry, no network calls
- **Deterministic** — same input, same report
- **Open source** — Apache-2.0
